# Mobile Screener JS Part

Run `npm install` to set project & download libraries.

## Running

* Q1: `node app-1.js`
* Q2: `node app-2.js`
* Q3: `node app-3.js`

## Tests

Run `npm test` for unit tests.

## JSHint

JSHint is installed as dev dependency.

```
node_modules/.bin/jshint app.js lib/
```
